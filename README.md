# Proyecto Clínica Saludable

La Clínica Saludable, ubicada en la calle Chacabuco #44 de Maipú, es un centro médico de tamaño pequeño que busca modernizar y optimizar sus procesos mediante la digitalización de sus servicios. Su propósito principal es ofrecer una atención más eficiente, ordenada y accesible a pacientes, médicos y personal administrativo. Para ello, la clínica está impulsando el desarrollo de un sistema integral de gestión de citas médicas, diseñado para automatizar procesos que actualmente se realizan de manera manual o presencial.

El sistema contempla funcionalidades clave como registro de pacientes, gestión de citas, administración de médicos, reportes básicos, y control de accesos según perfiles de usuario. La plataforma considera la experiencia de distintos actores, como pacientes frecuentes e invitados, médicos, administradores y secretarias. Asegurando que cada uno cuente con vistas y permisos ajustados a sus funciones.

Este proyecto incorpora buenas prácticas de seguridad (cifrado de contraseñas, control de roles, estándares OWASP), un diseño de base de datos robusto, formularios detallados para la captura de información clínica relevante y herramientas de retroalimentación para pacientes, especialistas y equipo de desarrollo.
